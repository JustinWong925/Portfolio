Cowmets
Inspired by the ESA's Rosetta probe and Philae Lander
An endless bullet-hell vertical scroller game.
This game was created solely in Javascript.  
Key features include: 
1. Being able to switching of weapons
2. On-the-fly upgrading
3. Resource system used for upgrading
4. Different enemy patterns both relative and not to the player character
5. A comet mining phase
