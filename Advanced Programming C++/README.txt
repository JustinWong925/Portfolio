CMPS 109: Advanced Programming
C++ programming

This course was mainly about C++ programming. 
There were 5 programs that covered some of the  different aspects of 
programming in C++.

1. Function Pointers as used in a shell program
2. Overloading and Operators creating a datatype of BigInt (numbers that would overflow standard datatypes)
	and creating operations such as addition/subtraction, multiplication/division on such numbers
3. Templates and Iterators, creating a listmap
4. Object Oriented Programming, Inheritance through the use of OOP and inheritance creating different
	objects, in this case shapes, that would be able to be drawn in OpenGL.
5. Client-Server, sockets creating a simple server that clients are able to connect to through sockets.
